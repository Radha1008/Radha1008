## Academic Projects @ IIT

### 🌐 Securing the IoT Landscape: Challenges and Strategies
> *ITMS-448/548 — Cybersecurity Technologies | Prof. Dr. Maurice Dawson*

A comprehensive research paper analyzing the cybersecurity challenges facing IoT ecosystems and proposing mitigation strategies.

**Key Topics:**
- **Threat landscape:** Botnets (Mirai), remote exploitation, data breaches
- **Core vulnerabilities:** Insecure APIs/interfaces, absent encryption, unpatched firmware
- **Real-world case studies:** Ring home security breach, 2016 Dyn DDoS attack
- **Security strategies:** Multi-Factor Authentication (MFA), end-to-end encryption (TLS/AES), automated firmware patch management

---

### 🔒 Exploring Multilevel Security (MLS) in Information Systems
> *Database Security Course | Illinois Institute of Technology*

A deep-dive into Multilevel Security models, their formal foundations, real-world applications, and implementation challenges.

**Key Topics:**
- **Security models:** Bell-LaPadula (confidentiality), Biba (integrity)
- **Access control mechanisms:** MAC, DAC, and RBAC
- **Real-world applications:** Military networks, government intelligence, healthcare (HIPAA), finance (GDPR)
- **Technical implementations:** SELinux, ACLs, PKI, AI-based anomaly detection for MLS enforcement
- **Emerging challenges:** Quantum computing threats, AI-based attacks, cloud scalability
